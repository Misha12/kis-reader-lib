# kis-reader-lib

Knihovna pro integraci čtečky karet s KIS aplikacemi.

## Překlad

### První kroky

```sh
npm install
```

### Běžný překlad

```sh
npm run build
```

### Vymazání souborů

```sh
npm run clean
```

#### Pozor

Nemaže soubory přeložené příkazem `npm run onefile`.

### Přeložení do jednoho souboru

```sh
npm run onefile
```
